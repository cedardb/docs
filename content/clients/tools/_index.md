GUI clients, CLI tools, and observability integrations that connect to
CedarDB over the PostgreSQL wire protocol. Unlike the language pages, these
are not tied to a specific programming language.

* [DataGrip](./datagrip) — JetBrains database IDE
* [DBeaver](./dbeaver) — cross-platform SQL GUI
* [Grafana](./grafana) — observability dashboards backed by CedarDB
* [psql](./psql) — the standard PostgreSQL command-line client
