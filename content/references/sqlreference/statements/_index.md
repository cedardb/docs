* Alter table
* Analyze
* Call
* Copy
* [Create database](createdb)
* Create function
* Create index
* [Create role](createrole) (create user)
* Create schema
* Create sequence
* Create table
* Create table as
* Create view
* Deallocate
* Delete
* Drop
* Execute
* Explain
* Insert
* Prepare
* Rename
* Select
* Set
* Show
* Transaction begin/end
* Update
* Vacuum
