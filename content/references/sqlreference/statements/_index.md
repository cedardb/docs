The complete list of supported SQL statements in CedarDB is as follows:

* Alter role
* Alter table
* Analyze
* Call
* Copy
* Create database
* Create function
* Create index
* Create role
* Create schema
* Create sequence
* Create table
* Create table as
* Create view
* Deallocate
* Delete
* Drop
* Execute
* Explain
* Insert
* Prepare
* Rename
* Select
* Set
* Show
* Transaction begin/end
* Update
* Vacuum
